HTML Тегтерімен Мәтін Форматтау


HTML негізі: HTML (Hyper Text Markup Language) интернетте беттер жасауға арналған тіл. HTML тілі жұп және жұпсыз тегтерді қоладну арқылы қүрылымдалады.

Мәтін Форматтау тегтерінің мысалы

Бұл қарайтылған мәтін және бұл көлбеу мәтін. Ал мынау асты сызылған мәтін.
Бұл сызылған мәтін, өшірілген мәтін, және енгізілген мәтін.

Маңызды: HTML тегтерін дұрыс қолдану веб-бет құрылымын жақсартады.

HTML физикалық және логикалық тегтері веб-дизайнда өте маңызды.

Бұл мәтін кішірейтілген.

Бұл мәтіннің ішінде ерекше маңыздылығы бар сөйлемдер қолданылған.
